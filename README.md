# Inferno Packs Shopify Theme

A GitHub-ready Shopify Online Store 2.0 theme for a wholesale repack website.

## Text customization

Open Shopify's theme editor, select **Inferno wholesale site**, and use the settings panel to edit:

- Announcement bar and logo words
- Navigation labels and buttons
- Hero headings, description, and accessibility text
- Lineup headings and copy
- Every pack name, subtitle, button label, and image alt text
- Every wholesale process step
- About section copy
- Contact headings, field placeholders, volume options, success message, and submit button
- Footer text

Pack cards and process steps appear as editable blocks in the section editor. The pack image file is selected by entering an existing filename from the theme's Assets folder.

## Shopify upload

Zip the theme contents so `assets`, `config`, `layout`, `locales`, `sections`, and `templates` are at the root of the ZIP. Then upload it from **Online Store > Themes > Add theme > Upload ZIP file**.
